# 📌 Project Overview

This project analyzes retail sales transactions to uncover valuable business insights such as:

- Revenue Trends
- Category Performance
- Customer Demographics
- Gender-wise Sales
- Age-wise Revenue
- Monthly Sales Trend
- Day-wise Revenue Analysis
- Quantity Sold by Category

The final output is a business dashboard suitable for portfolio presentation.

# 💡 Key Insights

### 📌 Total Revenue

- Total company revenue: **456,000**

### 📌 Best Performing Category

- Electronics generated the highest revenue.

### 📌 Customer Demographics

- Female customers generated slightly higher revenue than male customers.

### 📌 Highest Revenue Age Group

- Customers aged **46+** contributed the highest revenue.

### 📌 Monthly Trend

- May recorded the highest revenue.
- September recorded the lowest revenue.

### 📌 Best Sales Day

- Saturday generated the highest revenue.

# 📁 Dataset

The original dataset is **not included** in this repository.

The notebook contains the complete analysis workflow.

Required dataset columns:

```
Transaction ID
Date
Customer ID
Gender
Age
Product Category
Quantity
Price per Unit
Total Amount
```

If you use another dataset with the same columns, simply update the CSV file path in the notebook.

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/<your-github-username>/Retail-sales-analysis.git
```

### Navigate to Project

```bash
cd Retail-sales-analysis
```

### Install Dependencies

```bash
pip install pandas matplotlib notebook
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```

Open

```
retail_sales_analysis.ipynb
```

Run all cells.
